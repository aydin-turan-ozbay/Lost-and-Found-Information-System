# Lost and Found Information System

A web-based Lost and Found Information System designed for university campus.

## What will it do?

- report lost items
- report found items
- suggest match items using matching algorithms
- manage the delivery process easily and safely

## Technology Stack

### Frontend
-	HTML
-	CSS
-	JavaScript

### Backend
-	PHP

### Database
-	MySQL

### Development Tools
-   Visual Studio Code

## Core Features

-   User login and register with Role-Based Access Control (RBAC)
-   Email-Verified Registration System
-   Listing Categorization of Lost and Found Items
-   Automated Matching Algorithm
-   Visual Proof Layer
